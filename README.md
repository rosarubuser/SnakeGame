# SnakeGame

## To run this gameï¼š
1. download the package on your computer
2. compile the package by cmake
    ```
    $ cmake .
    $ make
    ```
3. run the executable SnakeGame

## Instruction
- use 'f' to speed up the snake speed, 's' to slow down.
- press 'q' to quit the game
